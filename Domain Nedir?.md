Domain adı, internet üzerindeki bir web sitesinin adresidir. IP adreslerinin sayısal formatını hatırlaması zor olduğundan, insanların kolayca anlayabileceği ve hatırlayabileceği metinsel karşılıklarıdır.
Örnek olarak:
```
www.google.com
www.amazon.com
www.twitter.com
```

### Domain adının yapısı genellikle şöyledir:
[alt domain].[ana domain].[üst seviye domain]
Örneğin,``` "blog.example.com" ```adresinde:

```"blog"``` alt domain
```"example"``` ana domain
```"com"``` üst seviye domain'dir

Diğer örnekler:

```tr.wikipedia.org```

```"tr" ```alt domain (Türkçe sürüm)
```"wikipedia"``` ana domain
```"org"``` üst seviye domain


```mail.google.com```

```"mail"``` alt domain (Gmail hizmeti)
```"google"``` ana domain
```"com"``` üst seviye domain



Domain adları, DNS (Domain Name System) aracılığıyla ilgili IP adreslerine çevrilir. Böylece kullanıcılar, karmaşık IP adresleri yerine akılda kalıcı isimlerle web sitelerine erişebilirler.
Üst seviye domainler çeşitli kategorilere ayrılabilir:

- Genel: .com, .net, .org
- Ülke kodları: .tr (Türkiye), .uk (Birleşik Krallık), .de (Almanya)
- Özel amaçlı: .edu (eğitim kurumları), .gov (hükümet kurumları)
